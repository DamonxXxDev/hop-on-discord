Have you ever wanted to link to a Discord server but never knew how to make it prominent enough or keep it light without loading too much crap? Hop on Discord is a simple tool to add a floating action button that links to your Discord server, which looks very similiar to the UI of Discord.

Setting up Hop on Discord is easy and can be done on any website so long as you have Custom HTML support (that does not block JavaScript or CSS).

• (1) First of all, visit this page. Copy the source code from there to a text editor (like Notepad, Atom, Notepad++, Sublime, any of those).

• (2) To change the server URL, go to the very bottom (at line 203) and replace the placeholder URL (https://discordapp.com/invite/0sg9HNylRzK4SnCr) with your own.

• (3) Here you can also change the text of the buttons or the introductory text (HANG OUT ON OUR DISCORD! TALK TO THE COMMUNITY BY TEXT OR ON VOICE CHAT!).

• (4) Once you are done, just add it to your website.

If you need extra help or want a new feature, create an issue on GitHub. Expect replies within a few days.

Hop on Discord

• Very compact (~7kb uncompressed)
• Simple to add & tweak
• Works everywhere
• No dependencies
• Minimal JavaScript
• No tracking
